# Sustento de Diseño – CrehanaWeb

## Principios de Diseño Visual
Se aplicó jerarquía visual con títulos grandes, colores contrastados y estructura por secciones claras.

## Arquitectura de Información
Se utilizó una sola página con navegación por anclas internas.

## Usabilidad
Botones visibles, navegación clara y formularios simples.

## Inclusive Design
Se utilizaron etiquetas ALT, ARIA-LABEL en inputs y contraste de colores adecuado.

## Tecnologías
HTML5, CSS3, JavaScript y Bootstrap vía CDN.

## SEO
Metadatos en header para posicionamiento.

## Segmentos
Estudiantes, diseñadores, fotógrafos y emprendedores digitales.
